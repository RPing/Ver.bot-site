Privacy Policy

Besides store your subscribed projects information in AWS DynamoDB, Ver.bot also store following items:

Slack: The channel id which Ver.bot interact with you.

Telegram: The chat id which Ver.bot interact with you.

Skype: The conversation id which Ver.bot interact with you.

The usage of these information is to notify you about new version information. All data in AWS DynamoDB will not expose to any third-party services or company.
