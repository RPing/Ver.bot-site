Collection of Information

Information from messaging platform

The information which will store in AWS DynamoDB:

Slack: The channel id which Ver.bot interact with you.

Telegram: The chat id which Ver.bot interact with you.

Skype: The conversation id which Ver.bot interact with you.

And of cource your subscribed project information, like project name and project author. Any other messages except above mentioned will not analyze or store to AWS DynamoDB. All data in AWS DynamoDB will not expose to any third-party services or company.
